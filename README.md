# Analytics Dashboard

## Project Overview

The Analytics Dashboard is a web-based dashboard designed to display and visualize analytical data. It provides insights into sales trends, key statistics, last orders, and top-performing platforms.

## Project Structure

### HTML Structure

- `index.html`: The main HTML file that structures the entire dashboard. It includes sections for the sidebar, body, and various components like sales trends, sales statistics, last orders, and top platforms.

### CSS Styles

- `style.css`: The main stylesheet file that defines the visual appearance and layout of the dashboard components.

### JavaScript

- `script.js`: The JavaScript file containing scripts for dynamic functionality, such as opening/closing the sidebar, handling user interactions, and managing the profile display.

### Project Assets

The `assets` directory contains various image files used for icons, logos, and graphical elements throughout the dashboard.

## Features

1. **Sidebar Navigation:**
   - Provides quick access to different sections of the dashboard.
   - Includes icons for categories, trends, profiles, boxes, discounts, info, and a theme switcher.

2. **Header:**
   - Responsive header design for both small and large screens.
   - Includes the dashboard logo, title, search functionality, user profile information, and notifications.

3. **Sales Trend:**
   - Visualizes sales trends over months with a bar chart.
   - Allows users to change the time range for a more detailed view.

4. **Sales Statistics:**
   - Displays key statistics related to total orders, total refunds, average sales, and total income.
   - Each statistic includes a percentage change compared to the previous months.

5. **Last Orders:**
   - Presents a table of recent orders with details like name, date, amount, status, and an option to view the invoice.

6. **Top Platforms:**
   - Highlights the top-performing platforms with their names, progress bars, total sales, and percentage changes.
